# Expense Tracker

A simple and responsive **Expense Tracker** web application built with **HTML, CSS, and JavaScript**.  
Track your expenses, filter them by amount and date, and sort them easily.

---

## Features

- Add expenses with **amount, date, and description**.
- **Filter expenses** by minimum/maximum amount and date range.
- **Sort expenses** by amount or date.
- Displays **total expenses** dynamically.
- Responsive and modern UI with smooth hover effects.
- Fully **client-side**, no backend required.

---

## Technologies Used

- **HTML5** – structure of the app  
- **CSS3** – styling, gradients, hover effects, shadows  
- **JavaScript (ES6)** – expense logic, filtering, sorting, DOM manipulation
